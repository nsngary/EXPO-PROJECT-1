# 在五個tabs中新增 stack ， EX:產品頁 -> A產品Detail

1. 將每個tabs 都變成資料夾

2. 每個tabs 資料夾中都必須含有:
    1. _layouts.tsx
    2. index.tsx

3. 其餘的檔案就是 stack

---

## _layouts.tsx

- 將其餘檔案名寫入layouts ，`<Stack.screen name="index" options.... 等`，將其變成 stack
*需要客製 options（title、headerStyle、presentation…）才需要寫 <Stack.Screen>*
- title 是每個分頁最上方的header
- 在`<stack></stack>` 中加入screenOptions來設定header字體等

```typescript
import React from 'react'
import { Stack } from 'expo-router'


const HomeStackLayout = () => {
  return (
    <Stack screenOptions={{
      headerTitleStyle:{
        fontFamily: 'NotoSerifTC_400Regular',
        fontWeight: '600'
      }
    }}>
      <Stack.Screen name="index" options={{ title: '最新消息' }} />
      <Stack.Screen name="gift-box" options={{ title: '禮物箱' }} />
      <Stack.Screen name="daily-check-in" options={{ title: '每日簽到' }} />
    </Stack>
  )
}

export default HomeStackLayout

```

## index.tsx

- index則是點擊每個tabs的`第一個畫面`
- 用 label 來決定每個 item.label 對映的檔案

```typescript
import React from 'react'
import { Pressable, StyleSheet, View } from 'react-native'
import { Link, Href } from 'expo-router'
import { Text } from '../../../components/StyledText'

type NavItem = { label: string; href: Href }

const Home = () => {
  const navItems: NavItem[] = [
    { label: '禮物箱', href: '/home/gift-box' },
    { label: '每日簽到', href: '/home/daily-check-in' },
  ]

  return (
    <View style={styles.container}>
      {/* <Text style={styles.title}>最新消息</Text> */}
      <Text style={styles.subtitle}>快速前往</Text>
      <View>
        {navItems.map((item) => (
          <Link key={item.label} href={item.href} asChild>
            <Pressable style={styles.item}>
              <Text style={styles.itemText}>{item.label}</Text>
            </Pressable>
          </Link>
        ))}
      </View>
    </View>
  )
}

export default Home
```
